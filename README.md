## ðŸ˜Š How to run

Before starting, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed on your machine.

```sh
- Clone this repo:
$ git clone https://github.com/marcelopajr/podcastr.git

- Enter directory:
$ cd podcastr

- Install dependencies:
$ npm install

- Launch the server:
$ npm run server

- Launch the app:
$ npm run dev
```

Have fun!
